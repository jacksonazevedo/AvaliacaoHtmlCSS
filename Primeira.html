    Projeto: PeopleTracker RFID.

    Atribuições:

    Aplicativo para android

    O objetivo deste aplicativo é realizar leituras RFID e enviar as tags lidas para o servidor MQTT. Foi desenvolvido utilizando o SDK Impinj e o modelo de chip Indy Impinj R2000.

    API RestFul

    Esta API tem os seguintes endpoints Empilhadeira, Manual e Portal. No qual cada recurso é possível cadastrar, alterar, pesquisar e excluir. A empilhadeira tem um filtro por id de empilhadeira. Todos os recursos obedecem o padrão HTTP e RestFul.

    Software de Gerenciamento RFID

    Este software possui três módulos: Leitor Generico, Configurar SmartPortal e Empilhadeira. O modulo leitor generico, após conectado com o servidor MQTT recebe leituras do leitor manual, portal ou SmartPortal e informa se a tag já foi lida ou não. No mesmo é possível escolher qual tipo de leitor deseja-se ler. No modulo Configurar SmartPortal é possível configurar o leitor alterando os parâmetros desejáveis. Já o modulo da empilhadeira consulta via API as tags cadastradas e compara com as tags lidas pela empilhadeira. Caso a tag lida exista, envia o endereço correspondente para a empilhadeira que leu.

     

    Palavras chaves: Java for Desktop, RFID, Leitura, Tag, SQLServer, ThingMagic.
